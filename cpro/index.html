Doing Assignments:
  1. For every problem write the solution in file code.c where the file problem.pdf is located.
  2. Run checking script script like this: ./check.sh Assignments/A1/BX. Replace the path of the question accordingly.
  3. If the script gives some permission error, run chmod +x check.sh.
  4. It's ethically wrong to see anything inside IO folder.
  5. Depending on your code, a test case can either pass or give TLE. Since the checker script is not perfect, it'll never give wrong rather will tell your output and expected output. It's then upto you to decide whether the output is same or different.
Labs:
